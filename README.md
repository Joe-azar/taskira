# Taskira

Taskira est une plateforme web de gestion de tickets, tâches et anomalies, conçue comme un projet full-stack moderne et réaliste.

## Stack technique

- Backend : Java 21, Spring Boot
- Frontend : Angular
- Base de données : PostgreSQL
- Authentification : JWT
- Conteneurisation : Docker

## Structure du projet

- `backend/` : API Spring Boot
- `frontend/` : application Angular
- `infra/` : fichiers Docker et configuration locale
- `docs/` : documentation, notes, captures

## Objectif

Construire une application de gestion de tickets crédible en environnement entreprise, valorisable sur CV et en entretien.
